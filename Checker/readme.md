-------------------
这是一个功能强大的本地评测器。

为什么这些事我不一次性搞定呢？当然是需求推进科技进步啦！~~都说了我懒~~

## Feature

- 评测（~~废话~~）并返回时间、内存使用信息
- 统计system call以及性能分析信息，浮点比较时统计精度
- 自动扫描并匹配数据以及待评测程序（最新编译的可执行程序）
- 支持从LOJ上获取时限、内存限制和数据
